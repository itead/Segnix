uint32_t analogWrite ( uint16_t  pin,
uint8_t  duty 
)

Output pwm with duty via the pin.

Parameters
pin- the number of the pin to output pwm.
duty- 0-255.
Returns
0 - success, 1 - failed.
Note
May be unstable when the duty be close to 0 or 255. A delay should be insert between two analogWrite calls.
Examples:
Dimmer.ino, and Fade.ino.
